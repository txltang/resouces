计算机工作原理教学设计

中职国家规划教材《计算机应用基础》

课程思政（课前德育3分钟）

同学们，计算机技术的发展体现了人类智慧的结晶。从第一台计算机ENIAC占地170平方米，到今天我们可以随身携带的智能手机，这是无数科学家和工程师共同努力的结果。

作为新时代的中职生，我们要学习他们勇于创新、严谨求实的科学精神，掌握好计算机技术，为将来建设科技强国贡献自己的力量。

教学目标

知识目标

了解计算机的基本组成结构和工作原理

能力目标

能够简单描述计算机处理信息的过程

素养目标

培养科学思维和逻辑分析能力

教学重点与难点

教学重点

  • 计算机五大部件及其功能
  • 计算机工作流程（输入-处理-输出）

教学难点

  • 控制器与运算器的协同工作原理
  • 存储程序概念的理解

知识复习（2分钟）

提问：上节课我们学习了计算机硬件组成，谁能回忆一下计算机有哪些主要硬件？

学生回答：显示器、键盘、鼠标、主机、打印机等

新课导入（3分钟）

情景创设：假设我们有一个"智能厨房"，厨师（CPU）接收订单（输入），从冰箱（存储器）取出食材，在灶台（运算器）加工，最后由服务员（输出设备）上菜。计算机工作过程与此类似，今天我们就来探索这个奇妙的过程。

讲授新知（20分钟）

一、计算机的五大部件

输入设备

计算机的"感官"

例子：键盘、鼠标、扫描仪

功能：将外部信息转换为计算机能理解的二进制代码

存储器

计算机的"记忆仓库"

例子：内存、硬盘

功能：存放程序和数据

运算器

计算机的"算盘"

功能：进行算术运算和逻辑运算

控制器

计算机的"指挥中心"

功能：指挥协调各部件工作

输出设备

计算机的"表达方式"

例子：显示器、打印机、音响

功能：将处理结果转换为人能理解的形式

二、计算机工作流程

输入

通过输入设备输入数据

→
存储

数据存入存储器

→
处理

控制器和运算器处理数据

→
存储

结果存入存储器

→
输出

通过输出设备展示结果

四、冯·诺依曼体系结构

提出者：美籍匈牙利数学家冯·诺依曼

核心思想：存储程序原理

特点：程序和数据一样存储在内存中，按顺序执行

板书设计

计算机工作原理
├─ 五大部件
│  ├─ 输入设备（键盘、鼠标）
│  ├─ 存储器（内存、硬盘）
│  ├─ 运算器（计算）
│  ├─ 控制器（指挥）
│  └─ 输出设备（显示器、打印机）
└─ 工作流程
   ├─ 输入 → 存储 → 处理 → 存储 → 输出
   └─ 例子：15+27=42

课堂练习（5分钟）

小组活动：请同学们分组，用肢体动作表演计算机处理"8×5+2"的过程，每组选一个代表解释表演的含义。

课堂总结（2分钟）

今天我们学习了计算机的五大部件和工作原理，知道了计算机如何像智能厨房一样处理信息。记住这个简单流程：输入→存储→处理→输出，这是所有计算机工作的基础。

教学反思
